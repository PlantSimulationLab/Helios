Write camera data for one or more bands to a JPEG image.

Parameters
[in]cameraLabel for camera to be queried
[in]bandsVector of labels for radiative bands to be written
[in]imagefile_baseName for base of output image JPEG files (will also include the camera label and a frame number in the file name)
[in]image_pathOPTIONAL: Path to directory where images should be saved. By default, it will be placed in the current working directory.
[in]frameOPTIONAL: A frame count number to be appended to the output image file (e.g., camera_thermal_00001.jpeg). By default, the frame count will be omitted from the file name. This value must be less than or equal to 99,999.
[in]flux_to_pixel_conversionOPTIONAL: A factor to convert radiative flux to 8-bit pixel values (0-255). By default, this value is 1.0, which means that the pixel values will be equal to the radiative flux. If the radiative flux is very large or very small, it may be necessary to scale the flux to a more appropriate range for the image.
