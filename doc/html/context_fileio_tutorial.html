Tutorial 4: File I/O

Helios can read and write several common file formats. In this tutorial we demonstrate how to import geometry from disk and how to save results for later use.

OBJ files are loaded with loadOBJ() and written with writeOBJ(). Geometry can be scaled and rotated on import.

Context context;
context.loadOBJ("../models/tree.obj", make_vec3(0,0,0), 5, nullrotation, RGB::green);
context.writeOBJ("tree_out.obj");

XML files store complete scene information, including geometry and timeseries data. Use loadXML() and writeXML() to read or write XML files.

context.loadXML("scene.xml");
context.writeXML("scene_copy.xml");

Text files are useful for exporting primitive data. The writePrimitiveData() function writes selected primitive data columns to a tab-delimited file.

std::vector<std::string> cols = {"primitive_ID", "temperature"};
context.writePrimitiveData("primitive_data.txt", cols, true);
